# WeCom Helper - 企业微信助手

> ✅ 企业微信官方 API 支持，合规自动化，无封号风险

企业微信智能客服助手，支持自动回复、人工介入、话术管理。

## 核心功能

- **自动回复**：关键词匹配，智能回复客户咨询
- **人工介入**：自动检测用户请求/负面情绪，一键转接人工
- **话术管理**：术语解释、禁用词过滤、客户专属配置
- **频率控制**：内置速率限制，符合平台规范
- **会话管理**：超时检测、情绪分析、历史记录

## 合规性

✅ **完全合规，可商用**

- 使用企业微信官方 API（非逆向）
- 符合《网络安全法》《数据安全法》《广告法》
- 数据本地存储，隐私保护

## 使用场景

1. **企业客服**：7x24 小时自动回复，复杂问题转人工
2. **销售支持**：产品咨询自动解答，线索自动分配
3. **内部办公**：员工问答自动化，HR/IT 支持
4. **会员管理**：VIP 客户专属话术，优先接入人工

## 依赖

- Node.js 18+
- 企业微信账号（需创建自建应用）
- 可访问外网的服务器（接收回调）

## 命令

```bash
# 启动企业微信助手
node index.js

# 初始化配置文件
node index.js --init

# 查看帮助
node index.js --help
```

## 配置说明

编辑 `config.json`：

```json
{
  "wecom": {
    "corpId": "你的企业 ID",
    "agentId": "应用 ID",
    "secret": "应用 Secret"
  },
  "humanHandoff": {
    "keywords": ["人工", "客服", "转人工"],
    "timeoutSeconds": 300,
    "agents": [
      {
        "name": "客服小王",
        "userId": "wangqiongzl",
        "available": true
      }
    ]
  }
}
```

## 人工介入机制

### 触发条件

| 条件 | 说明 | 优先级 |
|------|------|-------|
| 用户请求 | 关键词：人工、客服、转人工 | 🔴 高 |
| 负面情绪 | 关键词：垃圾、骗子、投诉 | 🔴 高 |
| 超时未解决 | 300 秒未解决问题 | 🟡 中 |
| 无匹配规则 | 无法自动回复 | 🟡 中 |
| 客户特殊要求 | 禁止讨论某话题 | 🔴 高 |

### 通知方式

- **Webhook**：推送至钉钉/飞书/企业微信机器人
- **消息通知**：发送企业微信消息给客服
- **邮件通知**：发送邮件告警（可配置）

## 话术管理

### 术语自动解释

配置专业术语，自动替换为通俗解释：

```json
{
  "terminology": {
    "SaaS": "软件即服务（Software as a Service）",
    "API": "应用程序接口"
  }
}
```

### 禁用词过滤

符合广告法要求，自动替换敏感词：

```json
{
  "forbiddenWords": {
    "最便宜": "性价比高",
    "最好": "优质",
    "第一": "领先"
  }
}
```

### 客户专属配置

为 VIP 客户配置专属话术和规则：

```json
{
  "clientSpecific": {
    "client_001": {
      "name": "XX 公司",
      "greeting": "尊敬的 XX 公司客户，您好！",
      "forbiddenTopics": ["价格", "竞品"]
    }
  }
}
```
